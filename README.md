# Ape Escape 4: Time Capture

**Made Ape Escape 4 because Sony's dumbass won't**

## Overview

A 3D platformer adventure game where you travel through time capturing mischievous apes using futuristic gadgets. Inspired by the original Ape Escape series, this fan-made game brings the classic formula to life with modern web technologies.

## Features

🎮 **Core Gameplay**
- 3D platformer exploration
- Time period-based levels (Prehistoric, Ancient Egypt, Medieval, Futuristic)
- Dynamic ape AI with unique personalities
- Gadget-based capture system

🛠️ **Gadgets**
- **Stun Rod** (1): Stun apes temporarily with electrical bursts
- **Time Hook** (2): Pull apes toward you across time
- **Capture Net** (3): Catch stunned or weakened apes
- **Ape Radar** (4): Highlight nearby apes for easy tracking

👾 **Apes to Capture**
- **Spike** - Prehistoric Era (Aggressive)
- **Buzz** - Prehistoric Era (Curious)
- **Pippo** - Ancient Egypt (Playful)
- **Kei** - Medieval Era (Cautious)
- **Yumi** - Futuristic Era (Mischievous)

## Controls

| Input | Action |
|-------|--------|
| **WASD** | Move around |
| **Space** | Jump |
| **Mouse** | Look around |
| **Mouse Click** | Use current gadget |
| **1-4** | Switch between gadgets |

## How to Play

1. **Explore** the level and locate apes
2. **Switch gadgets** based on ape behavior (Press 1-4)
3. **Stun** apes with the Stun Rod to make them vulnerable
4. **Capture** stunned apes with the Capture Net
5. **Collect all apes** to complete the level

## Technologies

- **Three.js** - 3D graphics rendering
- **JavaScript** - Game engine logic
- **HTML5/CSS3** - UI and styling
- **WebGL** - Hardware-accelerated graphics

## Game Strategy Tips

💡 **Pro Tips**
- Use the Ape Radar to locate harder-to-find apes
- Stun apes first before attempting capture
- Different apes require different strategies
- Watch ape movement patterns to predict their behavior
- Use platforms to gain high ground advantage
- Manage gadget cooldowns efficiently
